## рҹ“Ғ Dataset Details

> **Source:** [Netflix Movies and TV Shows till 2025 вҖ“ Kaggle](https://www.kaggle.com/datasets/bhargavchirumamilla/netflix-movies-and-tv-shows-till-2025) вҖ” Dataset includes detailed information on Netflix content (movies and TV shows). The two datasets were cleaned and appended into a single dataset for analysis.

**вҡ пёҸ Important Note on Genre Data:**
The genre column was split from multi-value cells into individual rows. This means **one title may appear multiple times** вҖ” once per genre. This is **intentional and by design**, not a data quality issue. All DAX measures account for this using `SUMX` + `VALUES` to prevent double-counting.

**вҡ пёҸ Financial Data Scope:**
Budget, Revenue & Profit data exists **only for Movies**. TV Shows have null values for financial fields. ROI is therefore **exclusively calculated for Movies** to maintain statistical accuracy.
